# FlashNotify 设计文档

## 应用概述
FlashNotify 是一个 Android 通知闪光灯助手，当收到指定 App 的消息通知时，自动启动闪光灯用户定义的闪烁，可自由定义持续次数

## 颜色方案
- 主色：深蓝黑 `#0F172A`（背景）
- 强调色：电光蓝 `#3B82F6`（主要按钮、激活状态）
- 成功色：`#22C55E`（运行中状态）
- 警告色：`#F59E0B`（权限提示）
- 表面色：`#1E293B`（卡片背景）
- 文字色：`#F1F5F9`（主文字）
- 次要文字：`#94A3B8`

## 屏幕列表

### 1. 主页面 (Home)
**内容：**
- 顶部：应用 Logo + 标题 "FlashNotify"
- 大开关卡片：主开关（启用/禁用闪光灯通知功能）
- 权限状态卡片：显示相机权限、通知访问权限状态
- 闪光灯设置卡片：闪烁次数、闪烁速度滑块
- 监听的应用列表（最多显示3个，点击"管理"进入应用选择页）
- 底部：测试按钮（手动触发一次闪光灯）

### 2. 应用选择页 (AppSelector)
**内容：**
- 搜索框
- 已安装 App 列表（图标 + 名称 + 包名 + 开关）
- 全选/全不选按钮

## 关键用户流程
1. 首次启动 → 请求相机权限 → 引导开启通知访问权限
2. 主页 → 打开主开关 → 选择要监听的 App → 等待通知
3. 收到通知 → 闪光灯闪烁 → 通知阅读后消失 → 闪光灯关闭

## 技术架构
- **通知监听**：Android 原生 NotificationListenerService（通过 Expo Modules 自定义模块）
- **闪光灯控制**：expo-camera CameraView + enableTorch prop（隐藏式 Camera 组件）
- **屏幕唤醒**：expo-keep-awake + Android WakeLock
- **数据持久化**：AsyncStorage 保存设置和应用列表
